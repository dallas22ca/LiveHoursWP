Add Live Chat Support for your website (the messages come to your phone a text messages).

== Description ==

LiveHours.co is a live chat support tool for small business owners. When you add a LiveHours live support chat to your website, your customers will be amazed at your instant, educated, world-class response... they won't have a clue that you're responding to them via text message. For more details, check out http://www.livehours.co.

This plugin makes it easy to install LiveHours on your Wordpress site. 
1. Install the plugin.
1. Enter your LiveHours token.

== Installation ==

1. Search for `LiveHours` in the 'Plugin Gallery' and use the one-click install *OR* [Download](https://github.com/dallas22ca/LiveHoursWP/downloads) the LiveHours.co plugin and Upload the `livehours` folder to the `/wp-content/plugins/` directory in your own WordPress installation.
